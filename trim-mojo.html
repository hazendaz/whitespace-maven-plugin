whitespace:trim

Full name:

com.github.hazendaz.maven:whitespace-maven-plugin:1.2.0:trim

Description:

Goal which trims whitespace from all requested extension files.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: process-sources.

Required Parameters

Name Type Since Description
<projectBasedir> File - Location of the file.
Default value is: ${project.basedir}/src.

Optional Parameters

Name Type Since Description
<extensions> String 1.1.0 File extensions to process.
Default value is: css,groovy,html,java,js,json,kt,md,properties,scala,sh,wsdl,xhtml,xml,xsd,yaml,yml.
<skip> boolean 1.2.0 Skip run of plugin.
Default value is: false.
User property is: whitespace.skip.

Parameter Details

<extensions>

File extensions to process.
  • Type: java.lang.String
  • Since: 1.1.0
  • Required: No
  • Default: css,groovy,html,java,js,json,kt,md,properties,scala,sh,wsdl,xhtml,xml,xsd,yaml,yml

<projectBasedir>

Location of the file.
  • Type: java.io.File
  • Required: Yes
  • Default: ${project.basedir}/src

<skip>

Skip run of plugin.
  • Type: boolean
  • Since: 1.2.0
  • Required: No
  • User Property: whitespace.skip
  • Default: false
