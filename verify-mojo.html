whitespace:verify

Full name:

com.github.hazendaz.maven:whitespace-maven-plugin:1.3.1:verify

Description:

Goal which detects whitespace in all requested extension files.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: process-sources.

Required Parameters

Name Type Since Description
<projectBasedir> File - Location of the file.
Default: ${project.basedir}/src

Optional Parameters

Name Type Since Description
<encoding> String 1.3.0 The character encoding scheme to be applied when filtering resources.
Default: ${project.build.sourceEncoding}
User Property: encoding
<extensions> String - File extensions to process.
Default: css,groovy,html,java,js,json,kt,kts,md,properties,scala,sh,toml,wsdl,xhtml,xml,xsd,yaml,yml

Parameter Details

<encoding>

The character encoding scheme to be applied when filtering resources.
  • Type: java.lang.String
  • Since: 1.3.0
  • Required: report.plugin.goal.no
  • User Property: encoding
  • Default: ${project.build.sourceEncoding}

<extensions>

File extensions to process.
  • Type: java.lang.String
  • Required: report.plugin.goal.no
  • Default: css,groovy,html,java,js,json,kt,kts,md,properties,scala,sh,toml,wsdl,xhtml,xml,xsd,yaml,yml

<projectBasedir>

Location of the file.
  • Type: java.io.File
  • Required: report.plugin.goal.yes
  • Default: ${project.basedir}/src
